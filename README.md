# walmart-stock-analysis
 Análisis y visualización del comportamiento histórico de las acciones de Walmart usando Python

## Descripción
Análisis exploratorio del comportamiento histórico de las acciones de Walmart 
utilizando Python. Incluye visualización de precios, medias móviles y volumen 
de transacciones.

## Tecnologías
- Python
- yfinance
- Pandas
- Matplotlib

## Análisis realizados
- Tendencia del precio de cierre histórico
- Variación porcentual diaria
- Medias móviles MA30 y MA90
- Volumen de transacciones

## Cómo ejecutarlo
pip install yfinance pandas matplotlib
python walmart_analysis.py
